# LogGate
An application for storing/searching logs, create an app, invoke endpoint with token, which includes appId, iat. review logs

Goal: made to store the logs in db/local for later debugging purpose

## Dependencies

- MongoDB
- Express
- NodeJs
- Redis
- Socket.IO
- Handlebars
